# 初二古诗文背诵游戏 · H5 发布说明

一个帮助初二学生背诵语文古诗文的纯静态网页小游戏，无需服务器、无需构建。

## 文件说明

```
初二古诗文背诵游戏/
├── index.html          # 页面结构
├── styles.css          # 样式
├── app.js              # 数据与游戏逻辑
└── README-H5发布.md     # 本说明
```

## 本地预览

直接用浏览器打开 `index.html` 即可：

```bash
# 方式一：双击 index.html
# 方式二：在文件夹内用 VS Code 的 Live Server 启动
# 方式三：用 Python 临时服务器
python -m http.server 8000
# 然后访问 http://localhost:8000
```

## 包含内容

- 18 首古诗文（唐诗、宋词、汉魏乐府等）
- 7 篇文言文（《三峡》《答谢中书书》《记承天寺夜游》《孟子三章》《与朱元思书》）

## 游戏模式

1. **下一句闯关**：给出上句，从 4 个选项中选出正确下句。答对得分，连击有加成。
2. **诗句排序**：把打乱的诗句/文句恢复原文顺序。
3. **作者朝代**：根据篇名选择作者或朝代。
4. **全篇目录**：浏览所有作品全文，可切换拼音显示。
5. **难句本**：自动记录答错的题目，支持单独练习。

## 数据维护

所有诗文内容都在 `app.js` 顶部的 `works` 数组中，按同样格式添加或修改即可：

```javascript
{
  id: "unique-id",
  title: "作品标题",
  author: "作者",
  dynasty: "朝代",
  type: "poem",          // poem 或 prose
  tags: ["标签1", "标签2"],
  contentRaw: [          // 每句一行，保留 inline 拼音
    "第一行内容pīnyīn，",
    "第二行内容。"
  ]
}
```

拼音会自动剥离；`contentRaw` 中的拼音在点击「显示拼音」后才会显示。

## 发布到网络

把本文件夹整体上传到任意静态托管即可：

- GitHub Pages
- Vercel / Netlify
- 腾讯云 COS / 阿里云 OSS
- 学校或个人服务器

## 注意事项

- 得分与难句本保存在浏览器 `localStorage` 中，换浏览器或清空缓存会丢失。
- 推荐在电脑、平板、手机上用现代浏览器（Chrome、Edge、Safari、微信内置浏览器）打开。
